# Accounting Acceptance Checklist

- لا توجد أخطاء `Binding` وقت التشغيل عند فتح شاشة اليوميات أو التقارير.
- يمكن حفظ سند قبض أو صرف أو تحويل كمسودة عبر `Ctrl+S`.
- يمكن حفظ السند وترحيله عبر `Ctrl+Enter`.
- يمكن إضافة سطر جديد في اليومية أو القيد الافتتاحي عبر `Ctrl+Shift+N`.
- إدخال مبلغ في `مدين` يمسح `دائن` في نفس السطر والعكس صحيح.
- السند المرحل لا يمكن ترحيله داخل فترة محاسبية مقفلة.
- إقفال الصندوق اليومي مع خيار القفل يحدّث تاريخ الإقفال على الشركة.
- `كشف الحساب` و`ميزان المراجعة` يعرضان نفس الفترة المختارة بدون اختلاف في الرصيد الختامي.
- ميزان المراجعة يسمح بفتح كشف الحساب مباشرة من صف الحساب.
- لا يظهر `ScrollViewer` أفقي داخل `DailyJournalDialog` أو `OpeningEntryDialog` أو `DailyCashClosingDialog` على العرض المتوسط.
